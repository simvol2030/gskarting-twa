┌──────────────────────────────────────────────────────────────────────────┐
│ DEVELOPMENT (5 minutes)                                                  │
└──────────────────────────────────────────────────────────────────────────┘

  Terminal 1:
  $ cd frontend-sveltekit
  $ npm run dev

  Terminal 2:
  $ cd frontend-sveltekit/cashier-electron
  $ npm install
  $ npm run dev

  ✓ Window opens at bottom-left corner
  ✓ Press F2 (earn), F3 (redeem), Esc (reset)

┌──────────────────────────────────────────────────────────────────────────┐
│ PRODUCTION BUILD (15 minutes)                                            │
└──────────────────────────────────────────────────────────────────────────┘

  $ cd frontend-sveltekit/cashier-electron
  $ npm run package:win

  Output: dist/Cashier-Loyalty-Setup-1.0.0.exe (~95MB)

┌──────────────────────────────────────────────────────────────────────────┐
│ CONFIGURATION (per store)                                                │
└──────────────────────────────────────────────────────────────────────────┘

  Create .env file in installation directory:

  BACKEND_URL=http://192.168.1.100:3000
  STORE_ID=1    # Change to 2, 3, 4, 5, or 6

┌──────────────────────────────────────────────────────────────────────────┐
│ KEYBOARD SHORTCUTS                                                       │
└──────────────────────────────────────────────────────────────────────────┘

  F2  = Earn Points (after sale complete)
  F3  = Redeem Points (before sale)
  Esc = Cancel / Reset to initial state

┌──────────────────────────────────────────────────────────────────────────┐
│ WINDOW BEHAVIOR                                                          │
└──────────────────────────────────────────────────────────────────────────┘

  Size:     1/3 screen width × 1/3 screen height
  Position: Bottom-left corner (0, 2/3 screen height)

  ✓ Always on top of other windows
  ✓ Cannot move
  ✓ Cannot resize
  ✓ Shows in taskbar
  ✓ Can minimize

┌──────────────────────────────────────────────────────────────────────────┐
│ TROUBLESHOOTING                                                          │
└──────────────────────────────────────────────────────────────────────────┘

  Problem: Window doesn't appear
  → Check taskbar, click to restore

  Problem: Can't connect to backend
  → Verify BACKEND_URL in .env
  → Test: curl http://your-backend:3000/api/health

  Problem: Shortcuts don't work
  → Click inside window to give focus

┌──────────────────────────────────────────────────────────────────────────┐
│ MULTI-STORE DEPLOYMENT                                                   │
└──────────────────────────────────────────────────────────────────────────┘

  1. Build once:      npm run package:win
  2. Copy to USB:     dist/Cashier-Loyalty-Setup-1.0.0.exe
  3. Install on each store
  4. Configure STORE_ID (1-6) per location
  5. Test connectivity
  6. Train cashiers on F2/F3/Esc
